# @streetstudio/api

**Primary domain responsibility:** API_Service — REST, WebSocket, and Webhook host built on StreetJS.

## Dependencies

- `@streetstudio/shared`, `@streetstudio/config`, `@streetstudio/database`, `@streetstudio/auth`, `@streetstudio/media`, `@streetstudio/processing`, `@streetstudio/notifications`, `@streetstudio/plugins`, `@streetstudio/analytics`
- StreetJS (`@streetjs/core`) — HTTP serving, routing, WebSockets, etc. via published public entry points.
