# @streetstudio/knowledge

Engineering knowledge for StreetStudio — the foundation of the knowledge-graph /
engineering-memory vision (see [`../../docs/PRODUCT.md`](../../docs/PRODUCT.md)).

`KnowledgeBase` handles transcript indexing (making recordings searchable within
scope), summaries (length-bounded, associated with a video), and documentation
links (validated, capped per video). This domain evolves independently of media
bytes/renditions.

## Dependencies

`@streetstudio/shared`, `@streetstudio/auth`, `@streetstudio/database`.
